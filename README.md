# Intune.HV.Tools.Gui

## Requirements

- PowerShell 5.1, or higher.

## How to use

Clone the whole project, including the submodules:

```
git clone https://github.com/pminnebach/Intune.HV.Tools.Gui --recursive
```

Run the `build.ps1` script.

Run the `run.ps1` script.
